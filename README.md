# Pacman JS

A classic Pacman game built with vanilla JavaScript and HTML5 Canvas.

> Forked from [tarantino07/pacman-js](https://github.com/tarantino07/pacman-js) with additional features and improvements.

## Demo

[Play Now](https://port-0-pacman-js-m8bka33ubb25cd8f.sel4.cloudtype.app/)

## Features

- Classic Pacman gameplay
- 4 ghost types with unique AI behaviors (Blinky, Pinky, Inky, Clyde)
- Boss Ghost spawns every 45 seconds
- Power Pellets to eat ghosts
- Hide ability to avoid ghosts temporarily
- Mobile joystick controls
- Sound effects and background music

## Controls

### Desktop
- Arrow keys or WASD to move
- Space to hide

### Mobile
- On-screen joystick
- HIDE button in center

## How to Play

1. Eat all the food pellets to win
2. Avoid ghosts or lose a life
3. Eat Power Pellets to turn ghosts blue and eat them
4. Use Hide ability to become temporarily invisible
5. Watch out for the Boss Ghost!

## Scoring

| Item | Points |
|------|--------|
| Food | 1 |
| Power Pellet | 50 |
| Ghost (combo) | 200 / 400 / 800 / 1600 |
| Boss Ghost | 500 |

## Run Locally

Simply open `pacman.html` in a web browser.
